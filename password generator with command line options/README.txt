Password generator with command line arguments
-h for help

-c 1 for charset 1 ("ABCDEFGHIJKLMNOPQRSTUVWXYZ" "abcdefghijklmnopqrstuvwxyz")
-c 2 for charset 2 ("0123456789" "ABCDEFGHIJKLMNOPQRSTUVWXYZ" "abcdefghijklmnopqrstuvwxyz")
-c 3 for charset 3 ("0123456789" "ABCDEFGHIJKLMNOPQRSTUVWXYZ" "abcdefghijklmnopqrstuvwxyz" "!+%&()=?*-")

-l [number] for specifying length

Example usage:

$ <program_name> -c 2 -l 256
